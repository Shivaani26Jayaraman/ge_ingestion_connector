# Python Math Utilities

A simple, lightweight Python repository providing fundamental mathematical operations, specifically focusing on generating Fibonacci sequences and calculating factorials efficiently.

This repository serves as a foundational project template and can be utilized to test repository indexing, custom search connectors, or basic GitHub Actions.

## Features

Fibonacci Sequence Generator : Iteratively generates a list of Fibonacci numbers up to a specified length `n` with O(n) time complexity.
Factorial Calculator : Efficiently computes the product of all positive integers up to `n` while handling edge cases (like zero and negative numbers) robustly.
